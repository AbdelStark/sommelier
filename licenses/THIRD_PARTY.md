# Third-Party Notices

This file records the license obligations attached to the third-party model,
dataset, and runtime packages that the Sommelier reference pipeline consumes.
It is checked by the release preflight (`sommelier release preflight`).

This file is not legal advice. Maintainers must re-verify each source card
before publishing any derived artifact.

## Base Model

- **Artifact**: `nvidia/Llama-3.1-Nemotron-Nano-8B-v1` (configured in
  `examples/config.full.yaml` under `model.base_model_id`).
- **Governing terms (per the model card)**: NVIDIA Open Model License
  Agreement, with the Llama 3.1 Community License Agreement applying because
  the model is built on Meta Llama 3.1.
- **Source card**: <https://huggingface.co/nvidia/Llama-3.1-Nemotron-Nano-8B-v1>
- **Recorded obligations for derived artifacts (adapters trained by this
  pipeline)**:
  - Comply with the NVIDIA Open Model License for use and redistribution of
    the base model and derivatives.
  - Comply with the Llama 3.1 Community License, including displaying
    "Built with Llama" prominently on distributed derived artifacts and
    including a copy of the Llama 3.1 license terms.
  - Do not use outputs to improve non-Llama large language models where the
    Llama 3.1 license forbids it.
- **Required notice text for derived model artifacts**: `Built with Llama`
- **Publication policy**: adapter weights are not published by default;
  publication requires maintainer verification of the obligations above
  (see release preflight).

## Dataset

- **Artifact**: `Salesforce/xlam-function-calling-60k` (configured in
  `examples/config.full.yaml` under `dataset.dataset_id`).
- **License (per the dataset card)**: CC-BY-4.0.
- **Source card**: <https://huggingface.co/datasets/Salesforce/xlam-function-calling-60k>
- **Recorded obligations**:
  - Attribute Salesforce as the dataset source in reports and derived
    artifact documentation.
  - Indicate whether changes were made to the data (Sommelier filters,
    deduplicates, and splits rows; it does not rewrite row content).
  - Note: the dataset card states the data was generated by APIGen; review
    the card's intended-use notes before redistribution of prepared splits.
